# Studentia

Studentia is a web app that integrates with Instructure Canvas to allow students to visualize and organize assignments in an intuitive way to help them with time management.

## Planned Features

- Load and sync assignments with canvas so you never miss an assignment
- Get reminders when assignments are due soon
- Arrange assignments based on when you plan to complete them
- Prioritize assignments by point values, grade weight, and due date

## Possible Features

- Use AI to estimate how long assignments will take to complete
